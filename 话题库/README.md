## 使用规范

### 标签数量
- 每篇笔记使用 **5-10个** 话题标签
- 标签应与内容主题高度相关

### 标签选择原则
1. **精准匹配**：选择与正文内容直接相关的标签
2. **热度适中**：包含热门标签和精准长尾标签
3. **领域聚焦**：优先选择餐饮、融合料理相关标签

### 标签组合示例

**菜品分享类**：
```
#融合料理培训 #菜品研发 #春季菜单 #Bistro菜单 #餐饮创业
```

**经验分享类**：
```
#融合料理教学 #菜单设计 #小酒馆菜品培训 #餐饮人 #开店经验
```

**行业分析类**：
```
#融合料理店 #餐饮趋势 #Bistro培训 #菜品创新 #餐饮管理
```

## 扩展方式

如需添加新话题：

1. 在 `餐饮服务/` 目录下创建新的 `.md` 文件
2. 按主题分类整理标签
3. 添加使用场景说明
4. 在主文档中更新目录

## 注意事项

- 话题标签会随平台趋势变化，建议定期更新
- 避免使用违规或敏感标签
- 禁止堆砌无关标签
- 标签应自然融入笔记内容
