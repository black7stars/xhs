# 笔记目录

本目录用于存储生成的小红书笔记文件。

## 目录结构

```
笔记/
├── .gitkeep              # 保持目录结构
└── [笔记标题]/           # 每篇笔记一个目录
    ├── 笔记.md           # 笔记正文（含配图建议）
    ├── 配图1.jpg         # 配图1
    ├── 配图2.jpg         # 配图2
    ├── 配图3.jpg         # 配图3
    ├── 配图4.jpg         # 配图4
    ├── 配图5.jpg         # 配图5
    └── 配图6.jpg         # 配图6（默认6张）
```

## 生成方式

### 单篇生成

使用命令：
```bash
/小红书
主题名称
[保存路径]
```

默认保存到：`笔记/[日期]/[笔记标题]/`

### 批量生成

使用命令：
```bash
/小红书批量生成
主题1
主题2
...
[保存路径]
```

### 自定义路径

在主题后添加路径（以`/`结尾）：
```bash
/小红书
春季菜单焕新
笔记/2026春季/
```

## 笔记文件结构

`笔记.md` 包含以下内容：

```markdown
**封面标题**
> 标题内容

**正文**
笔记正文内容...

**话题标签**
> #标签1 #标签2 ...

---

**配图建议**

### 配图1
详细配图建议...

### 配图2
...

---

**Banana提示词**

### 配图1 - [菜名]
**正向**：[提示词]
**负向**：[提示词]
**参数**：[参数]
```

## 使用规范

1. **禁止手动修改**：笔记文件由系统自动生成
2. **图片后处理**：如需调整，使用 `/小红书-图片后处理` 命令
3. **定期归档**：建议定期将历史笔记归档到子目录

## 注意事项

- 本目录已通过 `.gitignore` 配置，不会提交到 git
- 仅保留 `.gitkeep` 文件以保持目录结构
- 生成的笔记文件保存在本地，请自行备份重要内容
