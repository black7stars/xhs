## 核心原则

### 人设（3要素）
- **年限**：餐饮8年
- **背景**：ALMA厨艺学院
- **人称**：第一人称"我"

### 内容结构（4步式）
1. **痛点**：3-5句话，充分展开场景
2. **观点**：3-5句话，**必须有具体例子**
3. **案例**：具体场景+数据+效果
4. **结尾**：互动型或夯实观点型

### 红线（触发即返工）
1. 痛点少于3句或超过5句
2. 观点无具体例子支撑
3. 照搬素材库原菜名
4. 出现"10年"/"蓝带"/"我们"/"一碟盐"
5. 跳过4步结构中的任何一步
