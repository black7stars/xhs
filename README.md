# 小红书餐饮融合料理笔记生成器

基于 AI 的小红书餐饮内容自动化生成工具，专注中高端餐饮菜品研发与融合料理培训领域。

---

## 快速开始

### 环境要求

- **Python**: 3.8+
- **Node.js**: 16+ (用于 opencode 框架)
- **API密钥**: 需要配置 Banana API 和 SiliconFlow API

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目仓库地址>
   cd xhs
   ```

2. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，填入你的 API 密钥
   ```

3. **安装依赖**
   ```bash
   # 如果使用虚拟环境
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或 venv\Scripts\activate  # Windows
   
   # 安装 Python 依赖（如有需要）
   pip install -r requirements.txt
   ```

### 配置说明

编辑 `.env` 文件，配置以下参数：

```bash
# Banana生图API配置
BANANA_API_KEY=your_banana_api_key_here

# SiliconFlow图片分析API配置
SILICONFLOW_API_KEY=your_siliconflow_api_key_here

# 路径配置（根据你的实际路径修改）
PROJECT_ROOT=/Users/yourname/workspace/xhs
```

## 核心原则

### 人设（3要素）
- **年限**：餐饮8年
- **背景**：ALMA厨艺学院
- **人称**：第一人称"我"

### 内容结构（4步式）
1. **痛点**：3-5句话，充分展开场景
2. **观点**：3-5句话，**必须有具体例子**
3. **案例**：具体场景+数据+效果
4. **结尾**：互动型或夯实观点型

### 红线（触发即返工）
1. 痛点少于3句或超过5句
2. 观点无具体例子支撑
3. 照搬素材库原菜名
4. 出现"10年"/"蓝带"/"我们"/"一碟盐"
5. 跳过4步结构中的任何一步

## 故障排查

### 常见问题

**Q: 命令无法执行？**
- 检查 `.env` 文件是否正确配置
- 确认 API 密钥有效
- 检查网络连接

**Q: 图片生成失败？**
- 检查 Banana API 配额
- 确认网络稳定
- 查看错误日志

**Q: 素材库图片不匹配？**
- 运行 `/图片素材迁移` 添加新素材
- 检查 YAML 文件格式

### 获取帮助

- 查看各目录下的 `README.md` 文件
- 阅读命令文件中的详细说明
- 检查 `.opencode/skills/` 中的技能文档
