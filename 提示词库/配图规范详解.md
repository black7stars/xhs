# 配图规范详解

详细规范配图建议格式和摆盘说明标准。

## 一、配图建议标准格式

每个配图必须包含以下字段：

```markdown
### 配图X
- **素材**：[图片素材库路径]
- **菜品名**：[重新命名的菜品名称]
- **核心主体**：[主体物+数量+位置分布]
- **视觉元素清单**：
  - [元素1]：[详细描述]
  - [元素2]：[详细描述]
  - [元素3]：[详细描述]
- **色彩结构**：[主色+辅色+点缀色+氛围]
- **拍摄视角**：[俯拍/平视/特写/角度]
- **光线**：[光源方向+质感+阴影]

**摆盘说明（关键）**：
- **盘子选择**：[材质+形状+尺寸+边沿]
- **摆盘逻辑**：[构图法则+视觉流动方向]
- **间距控制**：[间距+分布特点]
- **装饰法则**：[装饰物+分布方式+效果]
- **留白艺术**：[留白位置+比例+作用]
- **整体风格**：[风格定位+场景契合]
```

## 二、字段填写规范

### 2.1 素材
- 必须指向图片素材库中的具体文件
- 格式：`素材库/图片素材库/[分类]/[文件名].jpg`
- 示例：`素材库/图片素材库/海鲜类/蟹肉塔_001.jpg`

### 2.2 菜品名
- 重新命名，禁止照搬素材库原菜名
- 使用"春日绯红蟹肉塔"格式（季节+颜色+食材+呈现方式）
- 高级感词汇：绯红、韵、佐、配、慢炖、低温、凝露等

### 2.3 核心主体
- 格式："[容器]中[数量]个[主体物]，呈[分布方式]，占据画面[位置]"
- 示例："白色极简盘中5个红色水滴形塔壳，呈不规则分布（右上2个+右下3个），占据画面右侧2/3"

### 2.4 视觉元素清单（必须逐项列出）

**结构**：
- 塔壳/主材：[材质+颜色+造型+尺寸]
- 内馅/配菜：[材质+颜色+形态]
- 装饰层：[装饰物1+分布]+[装饰物2+分布]
- 容器：[盘子类型+留白描述]

**示例**：
```markdown
- 塔壳：甜菜根天然红色外皮，叶片状造型，高度约2-3cm
- 内馅：蟹肉沙拉（灰白色质地），表面微隆起
- 装饰层：黄色小花碎密集点缀于塔壳表面 + 绿色圆形叶片（旱金莲叶，每塔2-3片，直径约2cm，叶脉放射状）
- 容器：纯白色哑光盘，大面积留白（左侧1/3空盘）
```

### 2.5 色彩结构
- 格式："[主色]（[对应元素]）+ [辅色] + [点缀色] —— [氛围描述]"
- 示例："红（塔壳）+ 白（盘面/蟹肉）+ 绿（叶片）+ 黄（碎花）—— 高饱和度对比，春日清新感"

### 2.6 拍摄视角
- 选项：垂直俯拍（90度）、45度斜俯拍、平视、特写微距
- 必须包含：镜头角度、透视关系、变形控制
- 示例："垂直俯拍（与水平面成90度），镜头正对盘面中心，无透视变形，呈现平面几何美感"

### 2.7 光线
- 格式："[光源类型]从[方向][照射方式]，[主体]表面[质感]，阴影[特征]"
- 示例："自然光从左上方柔和照射，塔壳表面有轻微光泽，阴影柔和不锐利"

## 三、摆盘说明详解（核心）

### 3.1 盘子选择
**必须包含**：
- 材质：纯白哑光/陶瓷/粗陶/玻璃/石板
- 形状：圆形/方形/不规则/深盘/浅盘
- 尺寸：直径（cm）
- 边沿：无边沿/极细边沿/宽边沿/特殊纹理

**示例**：
"纯白哑光圆形盘，直径约25-28cm，无边沿设计或极细边沿，最大化呈现盘面"

### 3.2 摆盘逻辑
**必须包含**：
- 构图法则：黄金分割/三分法/中心对称/对角线/不对称平衡
- 主体位置：占据画面比例+具体方位
- 视觉流动：引导线方向（左上→右下/左下→右上等）

**示例**：
"黄金分割构图，5个塔壳呈'右上聚集+右下延伸'的不规则群组，形成视觉流动线（从右上向左下对角线方向）"

### 3.3 间距控制
**必须包含**：
- 间距数值：X-Ycm
- 分布特点：等距/渐变/聚集+分散
- 视觉效果：独立性/群组感/层次感

**示例**：
"塔壳之间间距2-3cm，既保持独立性又形成群组感，避免平均分布的死板感"

### 3.4 装饰法则（最详细）
**必须包含**：
- 装饰物名称：旱金莲叶/三色堇/微型香草/食用花/粉末
- 数量/密度：每塔X片/密集覆盖/稀疏点缀
- 分布方式：均匀/随机/对称/放射状
- 效果描述：点状肌理/自然生长/几何排列

**示例**：
```markdown
- 旱金莲叶：每塔2-3片，叶片方向错落有致，部分叶片边缘轻微重叠，模拟自然生长状态
- 黄色碎花：密集但均匀覆盖，形成"点状肌理"效果，填补蟹肉与叶片间的空白区域
```

### 3.5 留白艺术
**必须包含**：
- 留白位置：左侧/右侧/上方/中心/边缘
- 留白比例：画面X/Y留白
- 视觉效果：呼吸空间/对比突出/画面平衡

**示例**：
"左侧1/3大面积纯白区域，与右侧色彩密集区形成强烈对比，突出主体同时保留呼吸空间"

### 3.6 整体风格
**必须包含**：
- 风格定位：法式精致感/日式极简/中式意境/北欧清新
- 对标标准：fine dining food presentation / michelin star presentation
- 场景契合：Bistro轻正餐/高端西餐/休闲brunch

**示例**：
"法式精致感（fine dining food presentation）+ 春日自然元素（花草装饰），符合Bistro轻正餐定位"

## 四、完整示例

```markdown
### 配图1
- **素材**：素材库/图片素材库/海鲜类/蟹肉塔_001.jpg
- **菜品名**：春日绯红蟹肉塔
- **核心主体**：白色极简盘中5个红色水滴形塔壳，呈不规则分布（右上2个+右下3个），占据画面右侧2/3
- **视觉元素清单**：
  - 塔壳：甜菜根天然红色外皮，叶片状造型，高度约2-3cm，酥脆质感
  - 内馅：蟹肉沙拉（灰白色质地），表面微隆起，细腻顺滑
  - 装饰层：黄色小花碎密集点缀于塔壳表面 + 绿色圆形叶片（旱金莲叶，每塔2-3片，直径约2cm，叶脉放射状）+ 微量柚子凝胶点缀边缘
  - 容器：纯白色哑光盘，大面积留白（左侧1/3空盘）
- **色彩结构**：红（塔壳）+ 白（盘面/蟹肉）+ 绿（叶片）+ 黄（碎花）+ 透明（凝胶）—— 高饱和度对比，春日清新感
- **拍摄视角**：垂直俯拍（与水平面成90度），镜头正对盘面中心，无透视变形，呈现平面几何美感
- **光线**：自然光从左上方45度角柔和照射，塔壳表面有轻微光泽和细微阴影，阴影边缘柔和不锐利，营造清晨窗光感

**摆盘说明（关键）**：
- **盘子选择**：纯白哑光圆形盘，直径约26cm，无边沿设计，盘面平滑无纹理，最大化呈现食物色彩
- **摆盘逻辑**：黄金分割构图（0.618比例），5个塔壳呈"右上聚集（2个）+右下延伸（3个）"的不规则群组，形成从右上向左下的对角线视觉流动线，引导视线自然移动
- **间距控制**：塔壳之间间距2-3cm，同组内间距2cm（紧密感），两组之间间距3.5cm（呼吸感），既保持独立性又形成群组感，避免平均分布的死板感
- **装饰法则**：
  - 旱金莲叶：每塔2-3片，叶片方向错落有致（左上/正上/右上各一），部分叶片边缘轻微重叠2-3mm，模拟自然生长状态，叶柄朝向画面中心
  - 黄色碎花：密集但均匀覆盖（每平方厘米约3-4朵），形成"点状肌理"效果，填补蟹肉与叶片间的空白区域，增加层次但不杂乱
  - 柚子凝胶：微量点缀于盘面前缘（3-4滴，直径约5mm），增加光泽感和味觉暗示
- **留白艺术**：左侧1/3大面积纯白区域（约8-9cm宽），与右侧色彩密集区形成强烈对比，突出主体同时保留呼吸空间，符合fine dining留白美学
- **整体风格**：法式精致感（fine dining food presentation）+ 春日自然元素（花草装饰），精致但不刻意，符合Bistro轻正餐定位，既有高级感又 approachable
```

## 八、批次一致性规则（批量生图强制遵循）

### 8.1 批次类型判定

**Type A: 多菜品批次 (Multi-Dish Batch)**
- **判定条件**：素材数量 > 1，或用户明确多菜品需求
- **一致性要求**：全部5项锁定整批一致
  - 拍摄视角（camera_angle）
  - 光线（lighting）
  - 盘子选择（plate_type）
  - 整体风格（overall_style）
  - 背景（background）

**Type B: 单菜品多视角 (Single-Dish Multi-View)**
- **判定条件**：素材数量 = 1，且生成数量 > 1，或为单篇笔记配图
- **一致性要求**：锁定3项，释放2项
  - ✅ 盘子选择（整批一致）
  - ✅ 整体风格（整批一致）
  - ✅ 背景（整批一致）
  - ❌ 拍摄视角（按多视角序列变化）
  - ❌ 光线（按多视角序列变化）

### 8.2 配置提取优先级

```
用户输入 > 主题推断 > 素材标签 > 默认值
```

1. **用户输入**：从自然语言中提取"俯拍"、"暖光"、"粗陶盘"等关键词
2. **主题推断**：Bistro主题→45度斜俯拍+暖光+粗陶盘+木质背景
3. **素材标签**：从首个素材YAML的plating_style/lighting/angle/background提取
4. **默认值**：45度斜俯拍、自然侧光、粗陶浅盘、Bistro轻正餐、复古木质桌面

### 8.3 Type A 批次规则（多菜品）

所有配图建议的6项摆盘说明中，以下字段必须**完全相同**：

| 字段 | 锁定值来源 | 示例值 |
|-----|----------|-------|
| 拍摄视角 | 批次配置 | 45度斜俯拍 |
| 光线 | 批次配置 | 自然侧光 |
| 盘子选择 | 批次配置 | 粗陶浅盘 |
| 整体风格 | 批次配置 | Bistro轻正餐 |
| 背景 | 批次配置 | 复古木质桌面 |

**禁止行为**：
- ❌ 配图1用俯拍，配图2用平视
- ❌ 配图1用暖光，配图2用冷光
- ❌ 配图1用白盘，配图2用粗陶盘

### 8.4 Type B 批次规则（单菜品多视角）

**锁定字段**（必须相同）：
- 盘子选择
- 整体风格
- 背景

**变化字段**（按以下序列变化）：

**拍摄视角变化序列**：
1. 配图1：俯拍（90度）- 整体摆盘展示
2. 配图2：45度斜俯拍 - 经典用餐视角
3. 配图3：平视 - 强调层次感
4. 配图4：特写 - 突出质感细节
5. 配图5：45度斜俯拍（另一侧）- 不同角度
6. 配图6：俯拍微距 - 细节放大

**光线变化序列**：
1. 配图1：自然光 - 真实还原
2. 配图2：暖光 - 温馨氛围
3. 配图3：侧光 - 强调质感
4. 配图4：顶光 - 均匀照明
5. 配图5：逆光 - 轮廓感
6. 配图6：柔光 - 均匀柔和

**说明**：视角和光线按顺序组合，确保每张图呈现不同侧面，同时保持品牌调性一致。
