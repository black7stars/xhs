# Banana提示词生成器

将配图建议（重点提取摆盘细节）转换为Banana模型提示词。

## API配置（已内置）

```yaml
Host: https://grsai.dakka.com.cn/v1/draw/nano-banana
Model: nano-banana-pro
Platform: GRS AI中转
Method: POST
Content-Type: application/json
```

## 输入

- 配图建议（必须包含完整摆盘说明）
- 菜品名称（用于识别，不加入提示词）

## 输出

- 正向提示词（英文，包含完整摆盘细节）

## 转换规则

### 1. 标准信息提取映射

| 配图建议字段 | 提取重点 | 转换为提示词 |
|------------|---------|-------------|
| 核心主体 | 主体物+数量+位置 | 主视觉描述 |
| 视觉元素清单 | 材质+颜色+形态 | 细节关键词 |
| **摆盘说明** | **盘子、逻辑、间距、装饰、留白** | **food presentation composition keywords** |
| 色彩结构 | 主色+辅色+点缀 | color palette |
| 拍摄视角 | 角度 | view perspective |
| 光线 | 光源 | lighting |

### 2. 【核心】摆盘专用映射表

这是转换的关键，必须完整提取：

| 摆盘说明子项 | 提取关键词 | Banana提示词转换 |
|------------|-----------|-----------------|
| **盘子选择** | 纯白哑光圆形盘/26cm/无边沿 | `white matte ceramic plate, 26cm diameter, smooth surface, minimalist tableware` |
| **摆盘逻辑** | 黄金分割/不规则群组/对角线流动 | `golden ratio composition, asymmetric arrangement, diagonal visual flow, clustered placement` |
| **间距控制** | 2-3cm/独立性+群组感 | `2-3cm spacing between elements, breathing room, clustered yet distinct arrangement` |
| **装饰法则** | 旱金莲叶2-3片/错落有致/自然生长 | `nasturtium leaves 2-3 pieces per tartlet, organic overlapping placement, natural growth pattern, edible flowers` |
| **留白艺术** | 左侧1/3留白/呼吸空间 | `left third negative space, large breathing room, minimalist negative space, high contrast composition` |
| **整体风格** | 法式精致感/fine dining/Bistro | `fine dining food presentation, michelin star presentation, bistro elegance, professional culinary photography` |

### 3. 提示词拼接公式

**标准结构**（按优先级排序）：

```
[质量标签], [主体描述], [盘子描述], [摆盘逻辑], [间距描述], 
[装饰细节], [留白处理], [视角], [光线], [色调], [风格], [技术参数]
```

**各段详细内容**：

1. **质量标签**（固定前缀，必加）：
   ```
masterpiece, best quality, photorealistic food photography, professional food presentation, 2k uhd, sharp focus, highly detailed
   ```

2. **主体描述**（从核心主体+视觉元素提取）：
   ```

   ```

3. **盘子描述**（从摆盘说明-盘子选择）：

   ```

4. **摆盘逻辑**（从摆盘说明-摆盘逻辑）：
   ```
   golden ratio asymmetric composition, diagonal arrangement from upper right to lower left, visual flow
   ```

5. **间距描述**（从摆盘说明-间距控制）：
   ```
   2-3cm spacing between tartlets, clustered yet distinct, breathing room between elements
   ```

6. **装饰细节**（从摆盘说明-装饰法则，最详细）：
   ```

   ```

   ```

   ```

9. **光线**（从光线）：
   ```

10. **色调**（从色彩结构）：
    ```

    ```

11. **风格**（从摆盘说明-整体风格）：
    ```
    fine dining food presentation, michelin star presentation, bistro elegance, fine dining aesthetic
    ```

12. **技术参数**（固定后缀）：
    ```

    ```

## 批量生成

当生成多张配图时（针对单一菜品的多视角策略），提示词差异化策略：

| 配图 | 差异化关键词 | 目的 |
|-----|------------|-----|
| 配图1 | 标准描述 | 基准图 |
| 配图2 | + "warm golden lighting" | 暖光氛围 |
| 配图3 | + "cool crisp lighting" | 冷光氛围 |
| 配图4 | + "45 degree angle shot" | 角度变化 |
| 配图5 | + "extreme close-up" | 特写细节 |
| 配图6 | + "different presentation arrangement" | 摆盘变化 |

## 注意事项

1. **摆盘是关键**：提取摆盘说明中的6个子项，全部转换为提示词
2. **无文字叠加**：确认不包含text overlay相关描述
3. **英文提示词**：全部转换为英文关键词
4. **逗号分隔**：使用英文逗号+空格分隔
