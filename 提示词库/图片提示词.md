# Banana提示词生成器

将配图建议（重点提取摆盘细节）转换为Banana模型提示词。

## API配置（已内置）

```yaml
Host: https://grsai.dakka.com.cn/v1/draw/nano-banana
Model: nano-banana-pro
Platform: GRS AI中转
Method: POST
Content-Type: application/json
```

## 输入

- 配图建议（必须包含完整摆盘说明）
- 菜品名称（用于识别，不加入提示词）

## 输出

- 正向提示词（英文，包含完整摆盘细节）

## 转换规则

### 1. 标准信息提取映射

| 配图建议字段 | 提取重点 | 转换为提示词 |
|------------|---------|-------------|
| 核心主体 | 主体物+数量+位置 | 主视觉描述 |
| 视觉元素清单 | 材质+颜色+形态 | 细节关键词 |
| **摆盘说明** | **盘子、逻辑、间距、装饰、留白** | **food presentation composition keywords** |
| 色彩结构 | 主色+辅色+点缀 | color palette |
| 拍摄视角 | 角度 | view perspective |
| 光线 | 光源 | lighting |

### 3. 提示词拼接公式

```

[装饰细节], [留白处理], [视角], [光线], [色调], [风格], [技术参数]
```

**各段详细内容**：

   ```
masterpiece, best quality, photorealistic food photography, professional food presentation, 2k uhd, sharp focus, highly detailed
   ```


5. **间距描述**（从摆盘说明-间距控制）：

| 配图 | 差异化关键词 | 目的 |
|-----|------------|-----|
| 配图1 | 标准描述 | 基准图 |
| 配图2 | + "warm golden lighting" | 暖光氛围 |
| 配图3 | + "cool crisp lighting" | 冷光氛围 |
| 配图4 | + "45 degree angle shot" | 角度变化 |
| 配图5 | + "extreme close-up" | 特写细节 |
| 配图6 | + "different presentation arrangement" | 摆盘变化 |

## 注意事项

1. **摆盘是关键**：提取摆盘说明中的10个子项（6项菜品呈现+2项摄影技术+2项物理环境），全部转换为提示词
2. **无文字叠加**：确认不包含text overlay相关描述
3. **英文提示词**：全部转换为英文关键词
4. **逗号分隔**：使用英文逗号+空格分隔
5. **禁止词检查**：生成提示词后必须执行禁止词扫描和替换

## 禁止词映射表（强制执行）

**生成提示词后必须执行以下替换**：

| 禁止词 | 强制替换为 | 替换原因 |
|-------|-----------|---------|
| `bistro` | `Modern Chinese fusion restaurant aesthetic (NOT traditional French bistro tavern; reimagined as refined small plates + curated wine selection + urban dining culture, contemporary minimalist interior design)` | AI误解为法式廉价小酒馆 |
| `tavern` | `contemporary dining establishment` | 易生成酒吧氛围 |
| `cozy atmosphere` | `clean minimalist presentation` | 情感模糊词 |
| `restaurant interior` | `clean table surface background` | 避免生成餐厅环境 |
| `dining scene` | `food presentation only` | 避免生成用餐场景 |
| `window` | `neutral background` | 避免生成窗户/临窗 |
| `patrons` / `diners` / `guests` | `[REMOVE]` | 禁止出现食客 |
| `waiter` / `chef` / `staff` / `person` / `people` | `[REMOVE]` | 禁止出现人物 |

**替换执行规则**：
1. 第5步生成提示词后立即扫描
2. 发现禁止词 → 立即替换为对应转述词
3. 替换后二次扫描确认无残留
4. 第5.5步API输入前最终验证

## 背景处理规范（菜单用图专用）

**默认背景**：纯桌子表面（非纯色/非环境）

```yaml
背景处理标准:
  类型: 纯桌子表面背景
  描述: "干净的桌面/台面表面，无任何其他物体或环境元素"
  材质选项:
    - 浅色木质纹理 (light natural wood grain)
    - 大理石纹理 (white/grey marble texture)
    - 混凝土纹理 (concrete texture)
  禁止元素:
    - 人物（食客/服务员/主厨）
    - 窗户/自然光线来源暗示
    - 其他桌椅/餐厅家具
    - 装饰物（花瓶/餐具/餐巾）
    - 环境氛围（灯光/阴影暗示空间深度）
    - 品牌标识/文字
  保留元素:
    - 仅有桌面/台面表面本身
    - 自然的桌面纹理（木纹/石纹）
    - 菜品投射的柔和阴影（用于锚定菜品）
  提示词核心: "clean [材质] table surface background, seamless texture, no other objects, only table visible, no people, no environment"
```
