## 一、4步结构详解

### 1.1 痛点描述（3-5句话）

**核心要求**：
- 必须有，且3-5句话
- 充分展开场景，不怕多
- 禁止排比句式"痛点1；痛点2；痛点3"

**4种处理方式**：

**方式A：痛点简述型**（最通用）
> 越来越多老板选择融合菜，但反馈不好干。花了大价钱装修出ins风，菜品却撑不起场景。想创新却没方向，试错成本不低。

**方式B：痛点+场景型**（节日/时令类）
> 春节赛道卷到没边！同质化菜品扎堆上桌，顾客看一眼就审美疲劳。想靠年夜饭冲业绩，要么毛利低得可怜，要么出餐慢到被催单。

**方式C：痛点+数据型**（行业分析类）
> 见过太多店开得轰轰烈烈，关得悄无声息。Bistro赛道卷成红海，同行转眼就能跟风模仿。到最后只剩装修内卷、低价厮杀，根本赚不到钱。

**方式D：痛点+疑问型**（专题型）
> 你店里的酱汁是不是总差点意思？跨菜系风味碰撞，没有好的融合酱汁。再优质食材也只是杂乱无章的拼凑，根本留不住回头客。

---

### 1.2 主厨观点（6选1，3-5句话）

**核心要求**：
- 3-5句话，充分展开
- 结构：现象/误区 → 核心观点 → 具体解释/例子
- 必须回应前文痛点
- 必须有具体例子支撑

**6种表达方式**：

**方式1："说实话"型**（直接切入）
- 适用：给出核心结论，回应痛点
- 衔接：直接回应前文痛点，给出解决方案方向

**方式2："我发现"型**（经验总结）
- 适用：分享多年经验得出的洞察
- 衔接：承接痛点，提出新的认知角度

**方式3："记住"型**（结论式）
- 适用：强调核心原则，memorable
- 衔接：总结痛点教训，给出关键法则

**方式4："其实"型**（转折深入）
- 适用：纠正常见误区，深入本质
- 衔接：指出痛点背后的认知偏差

**方式5："说到底"型**（本质剖析）
- 适用：剖析本质问题，直击核心
- 衔接：从表面痛点深入到根本原因

**方式6："回过头看"型**（复盘视角）
- 适用：用成功案例总结规律
- 衔接：从失败案例转向成功经验的提炼

**观点写作示例**：

❌ **避免（过于简略）**：
> 做了这么多年研发，我发现专业的融合料理菜单研发不是简单堆砌菜品，而是要有清晰的菜品结构逻辑。

✅ **推荐（充分展开）**：
> 做了这么多年研发，我发现融合菜菜单最容易陷入两个极端：要么是什么都有的大杂烩，要么是生硬的"中西拼盘"。其实专业的菜单研发不是简单堆砌菜品，而是要有清晰的结构逻辑——每道菜都要回答三个问题：它为什么出现在这里？能给客人带来什么价值？是否符合餐厅的整体定位？比如用「春涧云耳」这样的高颜值小食做获客款负责引流，用「琥珀牛肋」这样的经典主菜做利润款保证营收，再用一道创意招牌菜做形象款建立记忆点，三者缺一不可。

---

### 1.3 真实案例（3选1）

**核心要求**：
- 具体场景（给什么类型的餐厅做的）
- 方向性数据（成本可控/定价空间可观）
- 效果反馈（复购率提升/客人拍照发圈）
- 避免：具体价格数字、具体配方细节
- 人称：必须使用"我"而非"我们"

**3种呈现方式**：

**方式A：详细案例型**（有具体菜品）
> 上周给一家Bistro做春季菜单，用本土春笋搭配伊比利亚黑猪肉，通过融合料理的呈现方式，复购率提升明显，客人拍照发圈效果特别好。

**方式B：数据案例型**（强调效果）
> 我给一家日式brunch进行了菜单升级，升级后运营效率提升25%，菜品受欢迎度提升40%。

**方式C：简述案例型**（一笔带过）
> 之前给一家融合料理餐厅做酱汁体系搭建，建立了标准化的配比和操作流程，出品稳定性大大提升。

---

### 1.4 结尾（2选1）

**方式A：互动型**（主要）
> 你店里准备上什么新菜？评论区聊聊。

**方式B：夯实观点型**（强调结论）
> 这套逻辑已经经过100+餐饮品牌落地验证，每道菜都配套标准化SOP。

## 二、素材组合详解

### 组合1：底层逻辑型
- **痛点**：简述型
- **观点**：我发现型
- **案例**：数据案例型
- **干货**：并列要点型
- **结尾**：夯实观点型

### 组合2：节日菜单型
- **痛点**：场景+痛点型
- **观点**：说实话型
- **案例**：简述案例型
- **干货**：菜单清单型
- **结尾**：互动型

### 组合3：经验分享型
- **痛点**：数据/现象型
- **观点**：我发现型
- **案例**：详细案例型
- **干货**：经验要点型
- **结尾**：互动型

### 组合4：专题型
- **痛点**：疑问+痛点型
- **观点**：记住型
- **案例**：简述案例型
- **干货**：框架概述型
- **结尾**：互动型

### 组合5：菜品分享型
- **痛点**：简述型（快速带过）
- **观点**：说实话型（融入案例中）
- **案例**：详细案例型（全文聚焦1道菜）
- **干货**：详细做法型
- **结尾**：夯实观点型

## 三、内容类型详解

### 类型A：教育干货型（并列要点）
- **适用**：行业分析、经验分享、避坑指南
- **结构**：1️⃣2️⃣3️⃣，每点有标题+解释
- **干货详细程度**：每点包含"观点标题+展开解释（说明why）+方向指引（不说具体how）"
- **字数**：400-500字

### 类型B：食谱教学型（菜单清单）
- **适用**：具体菜品制作、配方分享
- **结构**：直接列菜名，分类展示
- **干货详细程度**：列出主要食材但不给具体克数，描述做法步骤但不给操作细节
- **字数**：200-300字

### 类型C：经验分享型（经验要点）
- **适用**：经验分享、破局思路
- **结构**：要点+口语化解释
- **干货详细程度**：给方向性建议，结合具体场景说明
- **字数**：300-400字

### 类型D：专题型（框架概述）
- **适用**：酱汁、摆盘、标准化等专题内容
- **结构**：给框架，简要说明
- **干货详细程度**：给框架和思路，不展开操作细节
- **字数**：300-400字

### 类型E：菜品研发型（详细做法）
- **适用**：具体菜品研发过程分享
- **结构**：食材+步骤
- **干货详细程度**：给食材和步骤，不给精确配方
- **字数**：300-400字

## 四、观点与干货的分工（重要）

### 核心原则
**例子应该在干货部分，而不是观点部分堆砌**。

### 观点部分（第2步）
**职责**：提出核心观点/方法论（3-5句话）

**内容**：
- 现象/误区描述
- 核心观点阐述
- 简要说明"是什么"和"为什么"
- **例子点到为止**（只提框架，不展开菜名）

**示例**：
> 做了这么多年研发，我发现融合菜菜单最容易陷入两个极端：要么是什么都有的大杂烩，要么是生硬的"中西拼盘"。其实专业的菜单研发不是简单堆砌菜品，而是要有清晰的结构逻辑——用"获客款+利润款+形象款"的三层结构，让每道菜都有明确角色。

### 干货部分（第3步后的要点）
**职责**：具体阐述"怎么做"，**每个要点必须有1个具体例子**

**内容**：
- 要点标题
- 核心原则说明
- **具体例子（含具体菜名）**

**示例**：
```
1️⃣ **获客款：高颜值+低门槛**
用「春日云耳」这样的创意小食做引流，拍照好看、价格友好，让客人愿意尝试并分享。

2️⃣ **利润款：经典+有溢价**
用「炭烤澳洲和牛肋」这样的招牌主菜做营收，本土食材配国际技法，成本可控的同时做出溢价空间。

3️⃣ **形象款：有记忆点+能传播**
用一道融合创意菜做品牌标签，不求每桌都点，但要让客人记得住、愿意推荐给朋友。
```

### 常见错误
❌ **错误**：观点部分详细列举3个菜名，干货部分只有原则无例子

✅ **正确**：观点部分提框架，干货部分的3个要点分别配1个例子

## 五、干货3层递进式写法（推荐）

基于运营实践优化的标准格式，确保内容既有深度又易落地。

### 标准格式

```markdown
**1️⃣ 标题层（≤15字）**
[类型标签]：[特征A]+[特征B]，[利益点]

**2️⃣ 观点层（1-2句，20-30字）**
底层逻辑/核心作用说明

**3️⃣ 例子层（1-2句，30-40字）**
用「具体菜名」...，[做法]...[经营场景效果]
```

### 优化前后对比

❌ **优化前（问题版本）**：
```markdown
1️⃣ **人气款：高颜值+低门槛**
用「山椒琉璃虾」这样的创意小食带动客流，拍照好看、价格友好，让客人愿意尝试并分享。
（缺少利益点标题，缺少观点阐述，直接给例子）
```

✅ **优化后（标准版本）**：
```markdown
1️⃣ **人气款：高颜值+低门槛，带动客流量**
人气款的核心作用是降低尝试门槛、带来新客。用「山椒琉璃虾」这样的创意小食，拍照好看、价格友好，让客人愿意尝试并主动分享，自然带动客流增长。
```

### 完整3点示例

```markdown
总结了3个菜单研发的破局要点：

1️⃣ **人气款：高颜值+低门槛，带动客流量**
人气款的核心作用是降低尝试门槛、带来新客。用「山椒琉璃虾」这样的创意小食，拍照好看、价格友好，让客人愿意尝试并主动分享，自然带动客流增长。

2️⃣ **利润款：经典+有溢价，保障高毛利**
利润款是营收主力，要用熟悉食材做出溢价感。像「炭烤澳洲和牛肋」，本土食材配西式技法，客单价高但成本可控，毛利自然可观。

3️⃣ **形象款：有记忆点+能传播，建立品牌认知**
形象款不求每桌都点，但要让人记住。比如一道融合中西的创意招牌菜，独特风味让客人吃一次就记住，主动推荐给朋友，形成口碑传播。
```

### 各类型框架优化

#### 类型A：教育干货型（增强版）
```markdown
结构：1️⃣2️⃣3️⃣，每点包含：
- 标题：标签 + 利益点（≤15字）
- 观点：底层逻辑（1-2句）
- 例子：具体做法 + 场景效果（1-2句）

字数：每点50-60字，3点150-180字
```

#### 类型C：经验分享型（增强版）
```markdown
结构：痛点场景 → 经验要点 → 正反案例
- 每个经验配：失败场景（避免）+ 成功做法（学习）
- 数据支撑：效果量化（复购率提升X%，营业额增长X%）

字数：每点50-60字
```

### 写作检查清单

- [ ] 标题≤15字，包含利益点
- [ ] 观点层说明底层逻辑/原因
- [ ] 例子层包含具体菜名（用「」标注）
- [ ] 说明经营场景效果（客流/毛利/品牌）
- [ ] 每点总字数≤60字

## 六、强制检查标准

### 4.1 人设一致性检查
- [ ] 年限是否为"8年"？（出现"10年""多年"等模糊表述即失败）
- [ ] 背景是否为"ALMA厨艺学院"？（出现"蓝带""法国"等其他背景即失败）
- [ ] 人称是否全程使用"我"？（出现"我们""咱们"即失败）
- [ ] 是否出现禁用品牌词？（"一碟盐"等即失败）

### 4.2 4步结构检查
- [ ] 是否完整包含4个步骤？
- [ ] 第1步（痛点）是否3-5句话？
- [ ] 第2步（观点）是否3-5句话？
- [ ] 第2步（观点）是否回应了前文痛点？
- [ ] 第2步（观点）后是否有具体例子支撑？
- [ ] 是否跳过任何一步？
- [ ] 顺序是否为1→2→3→4？

### 4.3 观点衔接检查
- [ ] 观点是否回应了前文的痛点？
- [ ] 观点与前文是否有割裂感？
- [ ] 观点后是否跟了具体例子支撑？
- [ ] 观点是否使用了6种方式之一？

### 4.4 菜品命名检查
- [ ] 是否照搬素材库原菜名？
- [ ] 菜名是否有高级感和创新性？
- [ ] 菜名是否过长（>10字）？
- [ ] 是否使用了简单堆砌？

### 4.5 违禁词检查

**高危词（必须修改）**：
- 直白价格："成本X块卖X""8块卖48"
- 绝对化："绝对""最好""第一""顶级"
- 夸张宣传："稳赚""包赚""必火"
- 医疗功效："养生""保健""疗效"
- 业绩夸张："翻倍""暴涨""暴增"

**中危词（建议修改）**：
- 营销话术："包教包会""加持""一站式解决"
- 导流诱导："加微信""私信我""进群"
- 虚假稀缺："限量""仅剩""手慢无"

## 五、红线清单

1. 痛点少于3句或超过5句话
2. 使用了排比句式"痛点1；痛点2；痛点3"
3. 观点未回应前文痛点（有割裂感）
4. 观点后无具体例子支撑（空洞）
5. 完全照搬素材库某一篇的结构
6. 照搬素材库菜品名称未重新命名
7. 与上一篇笔记结构相似度>50%
8. 连续2篇使用相同观点表达方式
9. 连续3篇使用相同创新角度
10. 跳过4步结构中的任何一步

## 六、完整示例（供参考）

**输入**：
```
/小红书
春季菜单焕新
```

**输出**：
```markdown
**封面标题**
> 利润⬆️60%，Bistro春季菜单这样换

**正文**

餐饮8年，ALMA厨艺学院毕业，见过太多餐厅在春季菜单上栽跟头。花大价钱请来的主厨，菜单却越做越杂，客人吃完根本记不住你家是做什么的。试错成本不低，换菜单反而换走了老客户。

做了这么多年研发，我发现春季菜单最容易陷入两个极端：要么盲目堆砌时令食材，要么是传统的照搬。其实专业的春季菜单不是追求花哨，而是找到"时令感+利润空间+易操作"的平衡点——用本土时令食材控制成本，通过融合料理技法提升档次，每道菜都要能出片、好操作、有利润。比如用春季新鲜白芦笋搭配深海鱼，低温慢烤保留本味，摆盘简洁有层次，成本可控但定价空间可观。

我给一家西餐厅做了春季菜单升级，用云南野生菌替代传统西式蘑菇做意式炖饭，保留了意式risotto的creamy口感，同时加入了本土菌菇的鲜香。复购率提升明显，很多客人冲着春季限定专门来打卡，拍照发圈的传播效果特别好。

总结了3个春季菜单的破局要点：

1️⃣ **不贪全、只聚焦**
菜单上什么都有，等于什么都没有。找准一个细分方向深耕，让客人一提到某个品类就想到你。

2️⃣ **时令食材+稳定出品**
春笋、荠菜这些时令食材确实吸睛，但必须配套标准化SOP，不能靠主厨手感。

3️⃣ **颜值要能发朋友圈**
春季菜要清新淡雅，摆盘要有呼吸感，客人愿意拍照分享才有传播力。

你店里准备上什么春季新菜？评论区聊聊。

**话题标签**
> #融合料理培训 #Bistro #春季菜单 #菜品研发 #餐饮创业

**配图建议**
> 封面主图：主厨手持春季时令食材（春笋/荠菜），背景明亮厨房
> 副图1：春季融合料理成品摆盘（留白构图）
> 副图2：菜单规划手稿或平板展示
```
