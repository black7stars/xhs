# 提示词库

本目录集中存储所有生成提示词、规范说明和参考素材，为命令文件提供详细的生成指导。

## 文档分类

### 核心规范

| 文档 | 用途 | 引用命令 |
|-----|------|---------|
| `生成规范详解.md` | 4步结构详解、素材组合、内容类型 | 小红书.md、小红书批量生成.md |
| `配图规范详解.md` | 配图标准、摆盘要求、Banana提示词规则 | 小红书.md、文生图.md、图生图.md |
| `图片提示词.md` | Banana模型提示词编写规范 | food-image-core skill |

### 创意指南

| 文档 | 用途 | 说明 |
|-----|------|------|
| `创新角度库.md` | 5大创新角度（食材/技法/酱汁/菜单/商业） | 避免重复，提供多样性 |
| `素材组合指南.md` | 5种素材组合方式 | 底层逻辑型、节日菜单型、经验分享型、专题型、菜品分享型 |
| `菜品命名规范.md` | 菜品创意命名公式 | 「季节+颜色+食材+呈现」 |

### 场景参考

| 文档 | 用途 | 说明 |
|-----|------|------|
| `痛点场景库.md` | 4种痛点描述方式 | 简述型、场景型、数据型、疑问型 |
| `封面标题库.md` | 封面标题模板 | 标题公式和emoji使用规范 |

## 使用方式

### 命令引用
命令文件通过相对路径引用提示词库文档：
```markdown
参见 `提示词库/生成规范详解.md` 第一章
```

### 快速查找
- 需要**结构规范** → 查看 `生成规范详解.md`
- 需要**配图指导** → 查看 `配图规范详解.md`
- 需要**创意角度** → 查看 `创新角度库.md`
- 需要**素材组合** → 查看 `素材组合指南.md`

## 文档维护

### 更新原则
1. 所有文档使用 Markdown 格式
2. 保持文档间引用路径一致
3. 更新时同步检查引用文档
4. 重大变更需更新版本日期

### 红线规范
- 禁止直接复制粘贴素材库原文
- 必须混合不同素材的表达方式
- 每篇笔记必须有至少1处创新
