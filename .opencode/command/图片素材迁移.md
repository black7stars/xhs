description: 自动分析、分类、迁移图片到素材库
agent: image-migration
---

扫描参考图目录，自动分析、创意命名、分类归档并迁移到素材库。

**参数解析规则**：
- 无参数，自动扫描 `参考图/` 目录

**前置检查**：
1. 检查 `参考图/` 目录是否存在
2. 扫描目录中的图片文件（jpg/png/jpeg）
3. 如无图片，提示用户先上传图片

**执行流程**：
1. 扫描 `参考图/` 目录，获取待处理图片列表
2. 调用 agent: image-migration
3. 传递参数：
   - source_dir: "参考图/"
   - target_dir: "素材库/图片素材库/"

**Agent 执行流程**：
- 第1步：环境检查（Python虚拟环境、API密钥、目录权限）
- 第2步：图片分析（调用 SiliconFlow API 分析每张图片）
- 第3步：创意命名（季节+颜色+食材+呈现）
- 第4步：自动分类（海鲜类/肉类/蔬菜类/菌菇类/其他）
- 第5步：生成YAML（6要素摆盘说明）
- 第6步：文件归档（复制到对应分类目录，自动编号）
- 第7步：自查修正（重复清理、YAML匹配、数量核对）
- 第8步：自动清理（清空参考图目录、生成报告）

**强制要求**：
- 必须完成全部8步
- 自动生成YAML标签文件
- 自动处理重名（添加序号_001, _002）
- 自查修正必须执行（重复清理、匹配检查）
- 清空参考图目录供下次使用

**使用示例**：
```
# 第1步：上传图片到 参考图/ 目录
# 第2步：
/图片素材迁移
```
