# 命令目录

本目录包含所有可执行的命令文件，用于生成小红书笔记、处理图片素材等操作。

## 命令列表

### 核心命令

| 命令文件 | 功能 | 使用场景 |
|---------|------|---------|
| `小红书.md` | 单篇笔记生成 | 生成一篇完整的小红书餐饮笔记（含配图） |
| `小红书批量生成.md` | 批量笔记生成 | 同时生成多篇笔记，自动去重 |
| `小红书-图片后处理.md` | 图片修复调整 | 对已生成笔记的图片进行修改、替换、重生成 |

### 图片处理命令

| 命令文件 | 功能 | 使用场景 |
|---------|------|---------|
| `文生图.md` | 文字生成图片 | 根据文字描述生成菜品图片 |
| `图生图.md` | 图片生成变体 | 基于上传图片生成风格变体 |
| `图片素材迁移.md` | 批量迁移素材 | 自动分析、分类、迁移图片到素材库 |

## 使用方法

1. **查看具体命令**：打开对应的 `.md` 文件查看详细用法
2. **执行命令**：按照文件中的输入格式使用
3. **查看示例**：每个命令文件都包含使用示例

## 依赖关系

- 所有命令都依赖 `.env` 文件中的API配置
- 图片相关命令依赖 `skill: food-image-core`
- 违禁词检测依赖 `skill: prohibited-words`

## 注意事项

- 命令文件遵循 Markdown 格式
- 文件头部的 YAML frontmatter 包含命令描述
- 实际执行通过 opencode 框架解析
