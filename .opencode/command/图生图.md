# 图生图

分析用户上传图片，自动入库后生成风格变体图。

## 输入格式

支持两种形式：

**形式A：仅上传图片（自动分析）**
```
/图生图
[数量，默认4张]
```
+ 上传图片附件

**形式B：上传图片+文字需求**
```
/图生图
6
想要冷色调版本，更多留白，适合冬季菜单
```
+ 上传图片附件

## 工作流程

调用 `skill: food-image-core` 完成以下流程：

### 第1步：图片分析
分析上传图片并生成YAML标签：
- 识别菜品内容、食材分类
- 分析摆盘结构、色彩、光线
- **自动保存**至素材库对应分类目录

### 第2步：需求融合（如用户提供）
将用户文字需求与图片分析结果融合：
- 保留原图核心元素（主体、摆盘结构）
- 叠加用户指定调整（色调/构图/风格等）

### 第3-5步：配图建议→提示词→生图
同文生图流程。

### 第6步：文件保存

**输出结构**：
```
AI生图/图生图/[日期]/
├── 配图1.jpg          # 生成的变体图
├── 配图2.jpg
├── ...
├── 原始图片_001.jpg    # 用户上传图备份
└── 生成记录.md        # 含分析结果和用户需求
```

## 支持的变体调整

用户可在需求中指定：
- **色调**：暖色→冷色、冷色→暖色
- **留白**：增加/减少留白比例
- **季节**：春季→冬季等季节氛围调整
- **光线**：自然光→暖光/冷光
- **风格**：Bistro→高端西餐等风格转换
