# 违禁词检测技能

## 工作流程

### 1. 识别检测需求
确认用户需要检测的内容类型：
- 单篇笔记全文检测
- 特定段落或表述检测
- 批量内容批量检测
- 特定词汇是否合规咨询

### 2. 扫描违禁词
按风险等级扫描内容：
- **高危词**（必须修改）：直白价格、绝对化用语、夸张宣传、医疗功效
- **中危词**（建议修改）：导流诱导、营销促销、虚假稀缺、金融敏感
- **低危词**（视场景调整）：未经认证、过度主观、互动诱导

### 3. 输出检测报告
为每个违禁词提供：
- 具体位置和原文
- 风险等级和原因
- 2-3个合规替代表述
- 优化后的完整文本

## 检测标准

| 风险等级 | 处理方式 | 示例 |
|---------|---------|------|
| 高危 | 必须修改 | "成本8块卖48" → "十几块成本做出大几十档次" |
| 中危 | 建议修改 | "加微信" → "评论区见" |
| 低危 | 视场景调整 | "内行人才知道" → "我的经验是" |

## 输出格式

```markdown
## 违禁词检测报告

### 高危词（X处）
- **位置**：第X行
- **原文**："XXX"
- **风险**：[原因说明]
- **建议**："[替代词]"

### 中危词（X处）
[同上格式]

### 低危词（X处）
[同上格式]

### 优化后文本
[完整优化版本]
```

## 注意事项

1. **价格敏感**：避免具体数字，使用模糊表述（"十几块"而非"8块"）
2. **绝对化**：多用个人感受，少用绝对承诺（"我觉得"而非"绝对"）
3. **宁可过度谨慎**：不确定时建议修改，避免触碰红线
4. **定期复查**：平台规则会更新，历史内容需定期检查

## 使用示例

**输入**：
```
这道菜成本8块卖48，绝对是全上海最好吃的意面！
```

**输出**：
```markdown
## 违禁词检测报告

### 高危词（2处）
- **位置**：第1行
- **原文**："成本8块卖48"
- **风险**：直白价格数字，易被判定营销号
- **建议**："十几块成本做出大几十的档次感"

- **位置**：第1行
- **原文**："绝对是"
- **风险**：绝对化用语
- **建议**："是我吃过"或"个人认为"

- **位置**：第1行
- **原文**："最好吃"
- **风险**：绝对化宣传
- **建议**："印象深刻的"或"值得一试的"

### 优化后文本
这道菜十几块成本做出大几十的档次感，是我吃过印象深刻的意面之一！
```
